Whisper Speech Recognition Model

Whisper is a general-purpose speech recognition model trained on a large dataset of diverse audio. It is a versatile multitasking model capable of performing multilingual speech recognition, speech translation, and language identification.

A Transformer sequence-to-sequence model is employed, which is trained on various speech processing tasks. These tasks include multilingual speech recognition, speech translation, spoken language identification, and voice activity detection.

The model interprets these tasks as a sequence of tokens to be predicted by the decoder. This innovative approach allows a single model to replace many stages of a traditional speech-processing pipeline. The multitask training format utilizes a set of special tokens that serve as task specifiers or classification targets.
